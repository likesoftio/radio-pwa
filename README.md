# Радио (PWA)

Мобильное устанавливаемое онлайн-радио на каталоге radio-browser.info. Полностью клиентское, без бэкенда.

## Разработка
- `pnpm install`
- `pnpm dev` — дев-сервер
- `pnpm test` — тесты (Vitest)
- `pnpm build` — прод-сборка с Service Worker и манифестом
- `pnpm preview` — предпросмотр прод-сборки (нужен для проверки PWA/SW)

## Стек
React + TypeScript + Vite, vite-plugin-pwa, Zustand, Dexie.js.

## Возможности
- Поиск и фильтры по названию / языку / жанру (каталог radio-browser.info)
- Избранное (локально, IndexedDB)
- Фоновое воспроизведение и управление с экрана блокировки (Media Session API)
- Название текущего трека из ICY-метаданных (с фолбэком на название станции)
- Установка как PWA, offline-запуск оболочки приложения
